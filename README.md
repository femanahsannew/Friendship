# Friendship Newsletter

A beautiful, responsive HTML newsletter about meaningful friendships and deeper connections.

## Overview

This newsletter explores the truth about friendship - not the comfortable version, but the honest one. It features the "Your Voice in My Headset - Friendship Episode" podcast discussion about building real, meaningful relationships.

## Topics Covered

- Why most relationships stay surface-level
- What real friendship costs you
- How identity shapes the way you connect
- Why strong friendships don't happen by accident

## Files

- `index.html` - The newsletter template (fully responsive)
- `README.md` - This documentation

## How to View

1. **Locally**: Open `index.html` in your web browser
2. **Online**: Host on GitHub Pages by enabling it in repository settings
3. **Email**: Send the HTML directly via email client
